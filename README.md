# 🧮 Simple Calculator

A menu-driven calculator developed using Python.

## 📖 Description

This project is a simple calculator that performs basic arithmetic operations such as addition, subtraction, multiplication, and division. 
The calculator uses functions and a menu-driven interface to provide a user-friendly experience.

## ✨ Features

- Addition
- Subtraction
- Multiplication
- Division
- Input Validation
- Menu-Driven Interface

## 🛠️ Technologies Used

- Python

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/mohansrimanthtalla/CALCULATOR.git
```

2. Navigate to the project folder:

```bash
cd CALCULATOR
```

3. Run the program:

```bash
python calculator.py
```
